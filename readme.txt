专为子比主题（Zibll Theme）设计的网盘链接批量管理工具，支持批量查找、替换、删除、导入导出及操作日志。

== Description ==

这是一个专为子比主题（Zibll Theme）用户设计的强大辅助插件。当您的网站拥有成百上千篇付费资源文章时，如果网盘链接（如百度网盘、蓝奏云）失效或需要更换域名，手动修改将是噩梦。本插件可以帮助您轻松批量管理所有文章中的下载链接。

**核心功能：**

*   **批量搜索与筛选**：按域名（如 baidu.com）或关键词快速查找链接，支持按发布时间筛选。
*   **高性能处理**：优化数据库查询逻辑，支持数万篇文章秒级搜索，包含分页功能。
*   **批量替换**：
    *   支持替换链接地址、显示名称、备注信息（提取码）。
    *   支持**正则表达式**替换（高级用户功能）。
    *   支持针对“选中项”或“全量搜索结果”进行操作。
*   **批量删除**：安全地批量移除失效或不需要的下载链接。
*   **操作日志**：自动记录所有批量操作和修改记录，支持查看操作人、时间和影响数量，防止误操作。
*   **导入/导出**：支持导出 CSV 备份，或从 CSV 批量导入链接到文章。
*   **即时编辑**：在列表页直接点击链接即可修改，无需进入文章编辑页。

**安全特性：**

*   严格的权限检查（仅限管理员）。
*   内置 CSRF 防护。
*   数据清洗与验证，防止 XSS 攻击。
*   卸载自动清理数据库残余。

== Installation ==

1. 上传 `zibll-download-link-manager` 文件夹到 `/wp-content/plugins/` 目录。
2. 在 WordPress 后台“插件”菜单中激活该插件。
3. 在后台“工具”菜单下找到“网盘链接管理”即可开始使用。
